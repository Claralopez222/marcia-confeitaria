# Site da Márcia Confeitaria — scaffold

Esta pasta contém um site estático de demonstração baseado na identidade visual fornecida.

O scaffold inclui:

- `index.html` — página principal
- `styles.css` — estilos básicos
- `assets/logo.png` — **adicione aqui o logo que você enviou** (nome do arquivo: `logo.png` ou `logo.svg`)

Como ver localmente:

1. Copie o logo enviado para `site/assets/logo.png`.
2. Abra `site/index.html` no navegador (duplo clique) ou rode um servidor estático, por exemplo:

```bash
cd site
python -m http.server 8000
# e abra http://localhost:8000
```

Quando você enviar os prints de referência, eu replico cada página exatamente dentro deste scaffold.
