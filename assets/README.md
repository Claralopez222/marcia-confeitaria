Place aqui os arquivos de imagem usados pelo site.

- `logo.png` ou `logo.svg`: logotipo principal (coloque o arquivo que você enviou aqui)
- `produtos/`: fotos dos produtos (nomes sugestivos: `bolo-aniversario.jpg`, `doces-finos.jpg`)

Observação: O scaffold usa `assets/logo.png` como referência no `index.html`. Se o seu arquivo tiver outro nome, atualize o `src` do `img` no `index.html`.
